¿Qué es el JSF ViewState? . JSF es una tecnología basada en controles que permite a los desarrolladores trabajar de una forma sencilla con controles muy complejos manteniendo el estado entre las diferentes peticiones HTTP (stateless). En muchas situaciones la forma de trabajar es muy transparente pero hay casos en los que es conveniente saber como el framework se comporta de forma exacta. Para ello es necesario conocer el concepto de  ViewState. Este concepto hace referencia a la estructura de controles que tenemos en una página concreta y a como JSF almacena su estado para posterior revisión.
