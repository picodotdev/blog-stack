Chocolatey quiere ser el Apt-Get de Windows, y es lo bueno de los proyectos libres, que si algo es bueno puedes cogerlo y portarlo a otras plataformas. La instalación es muy simple, al mas puro estilo GNU/Linux. Abrimos “Simbolo de sistema” con permisos de administrador y pegamos la siguiente linea: @powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))" && SET PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin Una vez instalado, cerramos la ventana y la volvemos a abrir. Ahora es tan simple como poner: choco install vlc adobereader Con esto se nos instalará Vlc y Adobe Reader. Aquí teneis una lista con el software disponible.
