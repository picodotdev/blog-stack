¿Quieres conocer las noticias y eventos de la Fundación de Software Libre? Entonces sigue leyendo La Fundación para el Software Libre o Free Software Foundation (FSF) es una organización creada en Octubre de 1985 por Richard Stallman y otros entusiastas del software libre con el propósito de difundir este movimiento. La Fundación para el software libre (FSF) se dedica a eliminar las restricciones sobre la copia, redistribución, entendimiento, y modificación de programas de computadoras. Con este objeto, promociona el desarrollo y uso del software libre en todas las áreas de la computación, pero muy particularmente, ayudando a desarrollar el sistema operativo GNU. Trustedsec ha anunciado el lanzamiento de una herramienta bastante interesante denominada PenTesters Framework (PTF). PTF es una herramienta desarrollada en Python diseñada específicamente para funcionar bajo Distribuciones basadas en Debian o Ubuntu, para convertirlas en distros ideales para las pruebas de penetración. El objetivo de la herramienta es la de instalar diversas herramientas para los Test de Penetracion, y que puedan ser actualizadas de forma sencilla en cualquier distribución. Todo esta organizado de manera que siga los lineamientos del Estándar de ejecución de Test de penetración, PTES ( Penetration Testing Execution Standard), eliminando todo lo que usualmente no se utiliza. Se trata de una vulnerabilidad del tipo “Buffer Overflow” que afecta la emulación del controlador del disco Floppy en el componente QEMU en los hipervisores KVM/QEMU y Xen. La vulnerabilidad fue registrada bajo CVE-2015-3456 y ahora es nombrado VENOM (virtualized environment neglected operations manipulation), es considerada de alto impacto.
