Por ejemplo, nos puede servir para transmitir por e-mail (o un chat) un archivo confidencial, para almacenar elementos privados en la nube, o para que alguien no se entere del contenido de un archivo. Guía rápida Para codificar un archivo: 1 $ openssl aes-256-cbc -in mi_archivo -pass pass:passw0rd -out mi_archivo.enc y para decodificar: 1 $ openssl aes-256-cbc -d -in mi_archivo.enc -pass pass:passw0rd -out mi_archivo Explicación El primer argumento es el algoritmo de cifrado/descifrado, podéis encontrar algoritmos como aes, des, base64 y muchos más.
