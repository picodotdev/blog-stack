El 2013 para Poesía Binaria

Escrito por Gaspar Fernández en la bitácora Poesía binaria, artículo original
Publicado el lunes, 30 de diciembre de 2013 a las 14:36 UTC
Etiquetado como estadisticas


Este es un pequeño resumen de lo que ha sido 2013 para este blog. Lo más visto, lo más comentado y algunas curiosidades:

Lo más visitado

  1. Recibiendo cadenas de texto completas con Arduino por USB (I)
  2. Formas de transformar un entero a cadena en C y C++
  3. Listar archivos dentro de un directorio o carpeta en C
  4. Comprobación de IP invertida / Reverse IP lookup
  5. Bucles for en BASH

En general, los artículos de C y C++ son más visitados que los de PHP o Java (aunque de Java no haya muchos todavía), y aparece un nuevo competidor, BASH, que normalmente también es muy visitado.

Lo más buscado dentro de Poesía Binaria

  1. imagemagick (gana por goleada)
  2. conio
  3. gtkmm
  4. gotoxy
  5. bash

Ahora tenemos imagemagick en la ecuación, es un dato muy curioso, aunque la búsqueda interna no se utiliza tanto. Sólo un 1.2% de los usuarios la ha utilizado.

Lo más buscado fuera de Poesía Binaria

  1. reverse ip lookup
  2. for en bash
  3. referencia mysql
  4. owncloud
  5. listas en java

Por lo que, los bucles en bash se buscan mucho, y contenidos sobre MySql, y cada vez más Owncloud (por cierto, la última versión está muy bien)

Navegadores más usados

  1. Google Chrome (54%)
  2. Mozilla Firefox (32%)
  3. Microsoft Internet Explorer (7%)
  4. Safari (5%)
  5. Opera (2%)

Más twiteado

  1. Magia en Bash, 50 ejemplos de una sola línea (o dos)
  2. Preparando nuestro entorno para Arduino sin Java
  3. Aventuras devolviendo la licencia de Windows que venía con un portátil
  4. 5 consejos para internacionalizar nuestros programas con gettext
  5. Ver la fecha de dmesg en formato humano

Más comentado

  1. Recibiendo cadenas de texto completas con Arduino por USB (I)
  2. Implementar un TPV con pasarela PASAT en PHP
  3. Aventuras devolviendo la licencia de Windows que venía con un portátil
  4. Creando un servidor que acepte múltiples conexiones en C
  5. Eliminar un elemento de un array en PHP

Enlaces entrantes

Quitando entradas de diferentes buscadores, agregadores y demás he de hacer una mención especial a:

  • Planetalinux
  • Código C
  • Ubuntizando

Sistemas operativos

  1. Windows 7 (46%)
  2. Linux (cualquier distribución, 22%)
  3. Windows XP (13%)
  4. Windows 8 (7%)
  5. OS X (4%)

Aunque el blog está especialmente dirigido a usuarios y desarrolladores bajo GNU/Linux, Windows sigue reinando en visitas, un 76% aunque también he de decir que sobre todo proceden de algún intercambiador de enlaces

Países

  1. España (42%)
  2. Mexico (18%)
  3. Argentina (9%)
  4. Colombia (6%)
  5. Chile (4%)

 

Otras curiosidades

En Facebook, sigue siendo muy visto cómo escribir en los muros desde nuestra aplicación, sigo recibiendo los mensajes de ejemplo del código muy a menudo

Las horas más visitadas son las 11-12 de la mañana y las 16-17 de la tarde. Coincide también con el lanzamiento de los artículos, siempre suelo lanzarlos después de las 10 de la mañana.

En todo el año, he tenido 68 nuevos Me gusta en la página de Facebook del sitio y 1 unlike.

Tengo que dedicar un poco más de tiempo a redes sociales, ya que de ahí vienen muchas visitas.

Ah !!! Por cierto, en el tema de dinero… bueno, tengo que decir que, ese tema no es que me haga millonario, como habréis visto, tengo muchos sistemas de publicidad, Adsense, Smowtion, Infolinks, ImpresionesWeb y alguno más, pero entre todos, en todo 2013 no he llegado a 10€, por lo que no es que el blog me dé de comer, pero me gusta
