La FSF y Debian unen fuerzas

Escrito por ochobitshacenunbyte en la bitácora Ocho bits hacen un byte, artículo original
Actualizado el martes, 09 de septiembre de 2014 a las 07:51 UTC
Etiquetado como blog-stack, bases-de-datos, sistemas-operativos, noticias, fsf, h-node, debian

La Free Software Foundation y el proyecto Debian unen fuerzas para ayudar a los usuarios del software libre a encontrar el hardware que necesitan. Ayer, día ocho de septiembre, anunciaron la cooperación para ampliar y mejorar h-node, una base de datos que tiene como fin identificar que dispositivos funcionan con los sistemas operativos completamente libres.

Actualmente existen bases de datos con los registros de hardware compatibles con GNU/Linux, en cambio h-node sólo identifica los que son sólo compatibles con software libre, y no necesitan software ni firmware privativo. La base de datos h-node lista tanto los componentes individuales, cómo tarjetas WI-FI y de vídeo, así cómo portátiles o laptops.

La información se obtiene de los usuarios que usan sistema libres y los prueban con el hardware de sus equipos. Antiguamente sólo se podían hacer pruebas con sistemas libres reconocidos por la FSF, aquí, lista que no incluía a Debian, ya que en Debian se puede utilizar el repositorios de software privativo non-free. La FSF en cambio reconoce el repositorio principal, main, cómo totalmente libre.

Según comenta John Sullivan, director ejecutivo de la FSF:

“A diferencia de otras distribuciones GNU/Linux, la instalación oficial de Debian por defecto usa únicamente software libre. Siempre que los usuarios no añadan repositorios privativos, la información sobre la compatibilidad se realiza desde software libre.  Tenemos muchas ganas de trabajar con Debian, para ayudar a los usuarios de software libre a encontrar el hardware que necesitan, y animar a las empresas fabricantes”

A comentado Lucas Nussbaum, líder del proyecto Debian.

“Al colaborar con h-node, Debian por primera vez tiene la oportunidad para unir esfuerzos con otras comunidades de software libre en el montaje de una base de datos de hardware que no requiere nada más que usar el repositorio main (el por defecto)”

“Debian tiene confía en los frutos de ésta colaboración. Invita a los miembros de su comunidad a contribuir con información de compatibilidad en h-node”

Sobre h-node

H-node fue iniciada por Antonio Gallo, que sigue siendo el líder del proyecto. La FSF actualmente proporciona la infraestructura y el apoyo. El software que usa h-node es software libre, usando la versión 3 de la Licencia Pública General GNU.

¿Cómo contribuir?

Los usuarios pueden contribuir, ya sea mediante el uso de un sistema operativo libre aprobado por la FSF, o con Debian, usando únicamente los repositorios de software libre, que son los que vienen por defecto (repositorio main)

Los desarrolladores y traductores pueden contribuir trabajando en el código fuente del sitio.  Más información sobre como contribuir aquí.

Sobre la Free Software Foundation

La Free Software Foundation (La Fundación para el Software Libre) fue fundada en 1985. y está dedicada a promover el derecho de los usuarios de las computadoras para poder usar, estudiar, copiar, modificar y redistribuir los programas informáticos. La FSF promueve el desarrollo y el uso del software libre, particularmente el sistema operativo GNU y las variantes GNU/Linux, y la documentación libre para el software libre. La FSF también contribuye a la difusión de la ética y las polítcias para libertad del uso del software, desde su sitio web, situados en fsf.org y gnu.org. Las donaciones para apoyar el trabajo de la FSF se pueden hacer aquí. Actualmente tiene su sede en Boston, en Estados Unidos.  Más información sobre la FSF, así como información importante para periodistas y editores, aquí.
