Instalar VMware-Tools en servidor virtual GNU Linux

Escrito por ochobitshacenunbyte en la bitácora Ocho bits hacen un byte, artículo original
Actualizado el viernes, 26 de septiembre de 2014 a las 07:10 UTC
Etiquetado como blog-stack, virtualizacion, kvm, comandos, sistemas-operativos, ubuntu, vmware, vmware-tools, vmware-workstation

Lo primero que nos damos cuenta cuando instalamos un servidor GNU Linux virtualizado, utilizando cualquiera de las plataformas VMware, es que debemos de instalar las herramietas VMware-tools. Hoy veremos como hacerlo.

En otros artículo hemos tratado de sistemas libres como KVM, creo que también es interesante que se traten otros sistemas que de una u otra manera tiene relación con GNU/Linux, aunque en parte o totalmente sean privativos.

Para el artículo he utilizado el sistema operativo virtual Ubuntu 14.04 LTS sobre VMware Workstation. Cómo podéis ver en la imagen:

Al seleccionar “Install VMware Tools” nos dirá que se instalará desde un CD en el sistema operativo virtualizado, le diremos que sí. Si tenemos algún medio utilizandose en la misma unidad de CD se desconectará.

Nos aparecerá el CD con un archivo comprimido dentro, los descomprimiremos donde más nos convenga:

Antes de llegar a la parte de la instalación de las herramientas, debemos de instalar los requisitos mínimos, que básicamente es el software necesario para compilar el código fuente.  Yo he usado un sistema basado en Debian como Ubuntu, así que todos sus derivados funcionan igual:

apt-get -y update
apt-get -y install build-essential
apt-get -y install linux-headers-$(uname -r)

Si todo ha ido bien, nos dirigimos al directorio del instalador de VMware.

Añadimos permisos de ejecución al fichero de instalación:

sudo chmod -x vmware-install.pl

Ahora ya podemos iniciar el programa:

sudo ./ vmware-install.pl
