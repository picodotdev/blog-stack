Instalar programas en Debian facilmente con tasksel

Escrito por ochobitshacenunbyte en la bitácora Ocho bits hacen un byte, artículo original
Publicado el jueves, 23 de octubre de 2014 a las 09:30 UTC
Actualizado el jueves, 23 de octubre de 2014 a las 09:38 UTC
Etiquetado como debian, blog-stack, ftp, sistemas-operativos, tasksel, programas, comandos, lamp, nfs, samba, mysql

En muchos de los artículos que he tratado en la web he tenido que instalar muchos paquetes de software, desde servidores LAMP, bases de datos, un servidor FTP o paquetes para servidores de archivos SAMBA o NFS.

Hoy hablaré de tasksel, una fantástica herramienta que nos facilitará las cosas cuando estemos instalando servicios en una máquina con GNU/Linux Debian.

Básicamente tasksel proporciona una sencilla interfaz a los usuarios que deseen configurar su sistema para realizar una tarea específica.

Cuando instalamos una distribución Debian por primera vez el programa que nos facilita las cosas es tasksel, pero también lo podemos invocar por consola a posteriori.

Si ejecutamos tasksel cómo usuario root nos mostrará una lista de grupos de paquetes a instalar.

En la imagen superior podéis ver varios, cómo Web server (LAMP), SQL database (MySQL), SSH server, entro otros.

Si queréis ampliar información sobre tasksel, podéis consultar la Wiki oficial, aquí.
