Joomla, segunda parada visitando los CMS

Escrito por ochobitshacenunbyte en la bitácora Ocho bits hacen un byte, artículo original
Actualizado el miércoles, 22 de octubre de 2014 a las 10:49 UTC
Etiquetado como blog-stack, cms, joomla, comunidad-open-source, debian

Entre los tres CMS más populares, se encuentra destacado Joomla.

Joomla es un sistema de gestión de contenidos o CMS, que nos permite ir más allá de las páginas estáticas, para generar sitios webs dinámicos. Tiene un potente panel de administración, quizás uno de los más completos.

Joomla, está desarrollado en PHP y liberado usando la licencia libre GNU GPL. Para su funcionamiento necesitamos un servidor con Apache2, PHP y MySQL.

Para el artículo he usado una máquinas virtual con Debian 7 Wheezy.

Le he instalado los paquetes de servidor web LAMP

El comando habitual es el siguiente:

apt-get -y update
apt-get -y install apache2
apt-get -y install php5 
apt-get -y install mysql-server mysql-common mysql-client
apt-get -y install php5-mysql libapache2-mod-php5

Una vez hecho esto podemos ir a la web oficial para descargar el software.

 

Una vez aquí nos iremos a la sección “Download“, justo en la parte superior.

Nos descargamos la versión 3.3 que es la recomendada para nuevos proyectos.

 

Lo descargamos dentro de la carpeta /var/www

cd /var/www
wget https://github.com/joomla/joomla-cms/releases/download/3.3.6/Joomla_3.3.6-Stable-Full_Package.zip

Accedemos a la terminal de MySQL, para crear las base de datos necesaria:

mysql -u root -p

Y ahora creamos y asignamos un nuevo usuario a la nueva base de datos:

CREATE DATABASE joomladb DEFAULT CHARACTER SET utf8 DEFAULT COLLATE utf8_general_ci;
GRANT ALL ON joomladb.* TO 'userjoomla'@'localhost' identified by 'contraseña';

Ya la tenemos creada la BBDD, ahora descomprimimos el ficheros. Antes instalamos el programa zip.

apt-get -y install zip
zip Joomla_3.3.6-Stable-Full_Package.zip

Con esto ya podemos reiniciar el servidor web

service apache2 restart

Si nos dirigmos a nuestro navegador web, ya podemos acceder: http://localhost/index.php

Aquí debemos indicar la información de nuestra web, respecto al nombre y al usuario administrador.

 

En la siguiente pantalla introducimos las credenciales de la BBDD

La próxima ventana nos da varias opciones, cómo cargar una BBDD de ejemplo, y también nos indica el estado actual de la instalación.

 

 

A continuación se iniciará la instalación.

Una vez finalizada ya podremos entrar al panel.

 

 

La configuración actual tiene cómo idioma el inglés, versión UK. Vamos a cambiarlo a castellano.

Justo en la columna de la izquierda, una vez estemos en el panel, tenemos la opción de cambiar idioma.

 

Nos dirigimos a “Install Language” y “Find Language” para buscar el castellano (Spanish)

Marcos el idioma “Spanish” y apratamos el botón de “Install“

Haciendo esto ya tendremos el idioma “Spanish” disponible.

 

Una vez realizado esto, salimos del panel.

Al volver entrar tendremos el idioma “Español” disponible.

Por cierto, cosa importante.

Me ha pasado que cuando quería entrar en la web, debía de escribir http://test.joomlaochobits/index.php, ya que si escribía sólo el dominio me devolvía un error.

Una de las posibles soluciones es eliminar el archivo index.html que va por defecto en el directorio /var/www
