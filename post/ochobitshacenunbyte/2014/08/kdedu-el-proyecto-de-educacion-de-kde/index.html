KDEdu, el proyecto de educación de KDE

Actualizado el jueves, 14 de agosto de 2014 a las 11:24 UTC
Etiquetado como kdedu, kturtle, klettres, kubuntu, educacion, cantor, edu-kde, kbruch, blog-stack, blinken, kgeography, kstars, kde, kalzium

Hoy descubrimos KDEdu, un proyecto colaborativo para el desarrollo de software educativo libre (bajo licencia GPL) para el escritorio KDE. El desarrollo está enfocado en crear productos para colegios, estudiantes y para los padres en casa, y para todas las personas que quieran ampliar sus conocimientos.

Un poco de historia

Según leemos en su web oficial, el proyecto KDE-Edu nació en julio de 2001, buscando el desarrollo de software educativo libre. Se creó un nuevo módulo en KDE, llamado kdeedu, que se unió al lanzamiento oficial de KDE 3.0 

Las aplicaciones

El proyecto desarrolla aplicaciones para diferentes franjas de edad y materias. Las franjas están divididas en tres categorías:

  • Preescolar
  • Escuela
  • Universidad

Y en temas:

  • Idioma
  • Matemáticas
  • Diversas
  • Ciencia

Instalando y probando

Para probar las aplicaciones he instalado en una máquina virtual un Kubuntu 14 de 32 bits. En esta distribución instalaremos las aplicaciones desde el centro de software “Muon Discover”, si usamos OpenSuse sería desde la aplicación YAST

Dejo nota de las que me han parecido más interesantes:

Para los más pequeños

KLettres – Aprender el alfabeto

Se trata de un programa pensado para aprender el alfabeto de un nuevo idioma y para aprender sílabas sencillas. Está pensando para niños a partir de los dos años y medio, o para adultos que quieren aprender lo básico de un idioma extranjero.

Por defecto podemos escoger entre Inglés, Inglés Británico, Castellano o Francés (hay 16 disponibles, aunque se pueden descargar idiomas adicionales)

Blinken - Memoriza colores

Se trata de un clón del famoso juego de los años ’70, que consistía en memorizar los colores, cuanto más colores memoricemos más puntos.

Para los chavales en edad escolar

Kalzium – La tabla periódica de elementos

Mucho más que la clásica tabla de elementos, podemos obtener información adicional de cada elemento y muchas más opciones como un editor de moléculas.

KBruch – Ejercicios con fracciones

Un pequeño programa para practicar ejercicios de fracciones y porcentajes.

Kgeography – Aprendremos geografía de manera divertida

Se trata de una herramienta de aprendizaje de geografía, que nos dará información adicional como capitales, banderas y otras.

KStars – Nuestro propio planetario en el escritorio

Se trata de un planetario de escritorio para KDE. Nos brindará una simulación gráfica del cielo nocturno, desde cualquier lugar de la tierra, a cualquier época del año, a cualquier hora. Para los amantes de la astronomía.

KTurtle – Un entorno de programación educativo

El programa quiere facilitar tanto como sea posible el aprendizaje de la programación. Todas las herramientas están disponibles en la interfaz de usuario. Usa un lenguaje de programación propio llamado TurtleScript.

Para universitarios

Cantor – Interfaz para software matemático

Una poderosa interfaz para matemática y estadística.

Si queréis saber más sobre las aplicaciones o ver todas ellas, sólo tenéis que ir a la página oficial del proyecto aquí.

Espero que haya sido de vuestro interés.
