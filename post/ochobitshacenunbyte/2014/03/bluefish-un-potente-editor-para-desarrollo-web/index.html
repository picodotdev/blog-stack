BlueFish, un potente editor para desarrollo web.

Escrito por ochobitshacenunbyte en la bitácora Ocho bits hacen un byte, artículo original
Actualizado el lunes, 10 de marzo de 2014 a las 13:21 UTC
Etiquetado como web, programacion, gnu-linux, css, c, bluefish, php, desarrollo, python, html

En castellano usamos frecuentemente la frase “Hay muchos peces en el mar”, según la circunstancia. Hoy hablaremos de un jugoso pez azul, que nos puede facilitar mucho las cosas, sobre todo para los desarrolladores web.

En GNU/Linux tenemos un sin fin de editores. Bluefish es un potente editor de texto enfocado al desarrollador.  Cómo características principales tenemos el resaltado de sintaxis, el apoyo a los proyectos, la sangría, la auto-realización, etcétera.  ¿Qué diferencia Bluefish de otros editores con funciones similares?

Partimos de la base que puede editar código para lenguajes como C, C++, Python entre otros. Pero dónde desmarca claramente es para el desarrollo web, ya que tiene soporte para HTML, CSS y PHP, además proporciona una lista todas las etiquetas HTML, y eso facilita mucha las cosas.  Aquí podemos obtener la lista de lenguajes soportados y características.

La última versión, que es la 2.2.5 salió el pasado 10 de febrero. Qué incluye mejoras en el sangrado, mejor resaltado de sintaxis, menor uso de memoria (cosa siempre a agradecer) entre otras mejoras. La nota de la versión la tenéis aquí.

Vamos a echarle un vistazo

Primero de todo lo tenemos que instalar. En distribuciones basadas en Debian, lo podemos instalar directamente con el comando apt-get o aptitude:

La instalación es muy sencilla:

aptitude install bluefish

La interfaz es bastante común a otros programas de edición. El programa está escrito en GTK. Cómo podemos ver, tenemos a la izquierda un espacio de navegación de carpetas, en la parte superior como es habitual el menú con todas las herramientas, con todas las etiquetas HTML y CSS, en la parte inferior información sobre las números de línea y posición.  Si hacemos click sobre la etiqueta se insertará automáticamente en el documento. También podemos resaltar que incluye ésta última versión las etiquetas HTML5, además de las estructura de raíz de las tablas y etiquetas.

En conclusión podemos decir que es muy buen editor de código, fácil de usar y muy potente, sobre todo para desarrollo web.
