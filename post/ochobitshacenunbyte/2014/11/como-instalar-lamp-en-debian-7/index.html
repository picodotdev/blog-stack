Cómo instalar LAMP en Debian 7

Escrito por socrates en la bitácora Ocho bits hacen un byte, artículo original
Actualizado el jueves, 13 de noviembre de 2014 a las 12:00 UTC
Etiquetado como mysql, blog-stack, sistemas-operativos, web, lamp, glamp, php5, debian

LAMP es un acrónimo de los servicios a instalar para permitir una infraestructura para servicios de Internet.

Lo componen Linux, Apache2, MySQL/MariaDB y Perl/Python/PHP5, también se le puede llamar GLAMP, al incluir GNU.

No voy a ir dando rodeos, y vamos a la faena:

Instalar Apache2

apt-get -y update && apt-get -y install apache2

Instalar el motor de base de datos, en este caso MySQL

apt-get -y install mysql-server mysql-common mysql-client

Cuando nos pida la contraseña simplemente la introducimos, no olvides apuntarla en algún sitio.

Instalar PHP5 y módulo Apache

apt-get -y install php5 libapache2-mod-php5

Aunque no es obligatorio, si que es recomendable, testear la instalación de PHP5

Lo podemos hacer de manera sencilla creando un archivo de texto, llamado info.php, dentro de /var/www

Lo creamos:

nano /var/www/info.php

Y lo editamos con:

<?php
phpinfo();
?>

Ya nos podremos ir a nuestro navegador web, y escribir: http://nuestra-ip/info.php

Instalaremos el módulo necesario para soporte de MySQL para PHP5

apt-get -y install php5-mysql

Es verdad que hay muchas más librerías, pero de momento no son necesarias.

Volvemos a reiniciar Apache2

service apache2 restart

Y ya lo tenemos listo : )

Ahora a crear proyectos!
