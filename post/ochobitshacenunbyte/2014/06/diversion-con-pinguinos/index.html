Diversión con pingüinos

Uno de los capítulos que más me gusta de la serie de The Bing Bang Theory es cuando Sheldon graba una pequeño vídeo llamado “Diversión con banderas“, pues bien, en éste artículo veremos una serie de comandos de consola, que nos harán pasar un buen rato.

Usaré para éste artículo un sistema Debian 7 Wheezy, así que el comando de instalación será apt-get sin el sudo delante, si usáis Ubuntu seguramente usaréis sudo apt-get y en distribuciones basadas en Red Hat, yum.

1. La Vaca Dice

apt-get -y install cowsay

Para que funcione directamente en cualquier parte de nuestro sistema, hemos de modificar la PATH:

PATH=$PATH:/usr/games
export PATH

Con la variable -f, podemos usar otro animal, como por ejemplo la mascota de Linux, tux.

2. Fortune

Dicho comando nos dará cada vez que lo llamemos una frase al azar.

Para instalarlo:

apt-get -y install fortune

Lo podemos combinar mediante una tubería o pipe con el comando cowsay:

fortune | cowsay -f tux

3. Para los fanáticos de los trenes

Steam Locomotive o sl, es un comando que nos devolverá el dibujo animado de una locomotora a vapor.

Para instalarlo:

apt-get -y install sl

4. Disfrutando del cine con telnet

Lo que no podías esperar del veterano programa telnet es que podrías disfrutar de una peli gracias a él, y menos de una como Star Wars, si ejecutáis el siguiente comando:

telnet towel.blinkenlights.nl

5. figlet o como crear banners de texto
Como ya de dice el titulo del apartado con el comando figlet podemos crear banners de texto en pantalla. Muchos demonios y servicios los utilizan.

Para instalarlo:

apt-get -y install figlet

Existen otros comandos parecidos para generar banners como toilet o banner.
