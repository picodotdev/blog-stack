Eclipse Luna ya está disponible

Escrito por ochobitshacenunbyte en la bitácora Ocho bits hacen un byte, artículo original
Publicado el jueves, 26 de junio de 2014 a las 07:53 UTC
Actualizado el jueves, 26 de junio de 2014 a las 07:53 UTC
Etiquetado como blog-stack, linux, noticias, mike-milinkovich, desarrollo, php, eclipse, java

La Fundación Eclipse acaba de anunciar el lanzamiento de la nueva versión, llamada Luna. Impulsada por una fantástica comunidad de desarrolladores, que han raelizado un trabajo duro con el resultado de más de 61 millones de líneas de código. Luna incluye muchas novedades como soporte para Java 8, muchas más bibliotecas y mejoras para el interfaz de usuario.

El trabajo de toda una comunidad

Mike Milinkovich, director ejecutivo de la Fundación Eclipse comenta en el anuncio oficial: “Felicidades a todos los desarroladores que hacen posible Eclipse. Es el trabajo duro de estos desarrolladores que hacen posible éste lanzamiento. El beneficio para la comunidad en general ha sido que Eclipse es un proveedor fiable y confiable de la tecnología de código abierto para la mayor industria de software “

Novedades destacadas

Entre las novedades se destacan:

  • Apoyo en Eclipse para Java 8. Soporte oficial de Java para las herramientas de desarrollo como Plug-in Development Tools, Eclipse Communication Framework o Xtext y Xtend.
  • Soporte para OSGi R6: Eclipse Equinox es ahora compatible con la especificación OSGi R6 recién publicada.
  • Mejoras en la interfaz de usuario WorkBench.
  • Actualización de las herramientas de desarrollo para PHP

Podéis saber más en la noticia de lanzamiento oficial (en inglés)
