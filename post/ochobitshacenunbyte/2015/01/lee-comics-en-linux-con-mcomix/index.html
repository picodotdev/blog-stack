Lee cómics en Linux con MComix

Escrito por davidochobits en la bitácora Ocho bits hacen un byte, artículo original
Publicado el jueves, 08 de enero de 2015 a las 10:24 UTC
Etiquetado como comic, blog-stack, programas, manga, mcomix, gnu, linux, debian

Para los fans de los cómics hoy hablaré de MComix. Un fantástico software, visor de imágenes, diseñado para leer cómics y manga.

Es la continuación del proyecto Comix, que quedó descontinuado en 2009.

Puede leer múltiples formatos como tar, zip, rar, 7zip, cb7, cba, entre muchos otros.

Está escrito en lenguaje de programación python y funciona tanto en GNU Linux, FreeBSD como Windows.

Instalación en Linux

Está disponible en los repositorios main de distribuciones como Arch Linux, Ubuntu y Fedora. Y también podemos compilar en programa desde el código fuente.

MComix en Debian

En Debian también la instalación es igual de fácil, simplemente debemos de escribir en la consola de comandos:

1

apt-get -y install comix

El propio instalador se encargará de las dependencias necesarias.

Una vez instalado el aspecto es el siguiente:

Os recomiendo la web digitalcomicmuseum.com, donde encontré el cómic de la imagen de más arriba. Todo el depósito es de dominio público.
