UberStudent lanza la versión 4.1

Escrito por davidochobits en la bitácora Ocho bits hacen un byte, artículo original
Publicado el lunes, 19 de enero de 2015 a las 09:55 UTC
Etiquetado como learning, blog-stack, sistemas-operativos, ubuntu, moodle, uberstudent, gnu, linux

El pasado viernes, la distribución basada en Ubuntu, enfocada a la comunidad educativa, que utiliza el escritorio XFCE, UberStudent, lanzó su versión 4.1

UberStudent está pensada para aprender y enseñar computación a nivel académico, en los niveles de educación secundaria y superior. Lleva incorporados una serie de programas y características pensados para enseñar y facilitar las tareas en todas las áreas académicas, incorpora las tecnologías Moodle.

La distribución busca ser mucho más que un simple sistema operativo, “UberStudent redefine fundamentalmente lo que significa ser un sistema operativo para la educación”, “se trata de un plan de estudios pedagógicamente coherente éxito académico integrado en una plataforma de aprendizaje fácil de usar, y con todas las funciones instalable.”

Diseñada para la educación

Aspira a ser una potente plataforma de aprendizaje, un sistema unificadado, para aprender, hacer y enseñar el éxito académico. Está diseñado para facilitar el dominio de las habilidades básicas, como las de investigación, escritura, el estudio y la autogestión.

Aplicaciones destacadas

El sistema incorpora de manera nativa un larga lista de aplicaciones, entre las que destacamos:

  • CherryTree. Programa que nos permite crear notas de forma jerárquica. (Más info aquí)
  • Freeplane. Una herramienta para organizar y estructuras la información. (Más info aquí)
  • Zotero. Aplicación que recolecta, gestiona, cita y comparte fuentes de investigación que encuentres por Internet. (Más info aquí)
  • LibreOffice. Incorpora la popular suite de ofimática.
  • Anki. Un sistema de aprendizaje espaciado inteligente y fácil de usar. Genial para memorizar material didáctico. (Más info aquí)
  • Artha. Diccionario basado en WordNet, que nos ayuda a buscar el significado de expresiones regulares. (Más info aquí)
  • bibme. Es un generador automático de bibliografías. (Más info aquí)
  • A nivel gráfico reune una serie muy interesante de programas: GIMP, Inkscape, Pinta, gtkam o PhotoPrint.
  • Respecto a las aplicaciones relacionadas con Internet: Firefox, Google Chromium, Thunderbird, Pidgin, Dropbox, gPodder o Deluge.
  • A nivel de multimedia incorpora: Imagination, Audacious, VLC, Totem Media Player o MediaInfo.

Podéis visitar su página web y descargar el DVD

La única pega que le veo es que la traducción del castellano todavía dista mucho de ser la deseada. A pesar de eso me parece una distribución muy potente, ideal para todos los miembros de la comunidad educativa, desde educadores hasta estudiantes.
