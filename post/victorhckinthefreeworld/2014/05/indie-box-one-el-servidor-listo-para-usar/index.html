Indie Box one el servidor listo para usar

Escrito por Victor Hck en la bitácora Victorhck in the free world, artículo original
Actualizado el domingo, 11 de mayo de 2014 a las 22:49 UTC
Etiquetado como blog-stack, gnu-linux, noticias

Con Indie Box one puedes tener tu propio servidor casero, sencillo de usar administrar y con todo lo que necesitas. Sencillo y potente y además lleno de software libre

¿Quieres tener tu propio servidor casero desde el que poder publicar en tu web mediante WordPress, tener tu nube privada de datos mediante ownCloud, tu propio servidor de correo, y mucho más? Y además quieres tenerlo sin tener que ser un Geek, para instalar todo ese software? Lo que estás buscando es Indie Box one.

Este proyecto propone hacer fácil la tarea de montar un servidor casero “out of the box“. Un servidor con:

  • 2Gb de Ram
  • Procesador Atom dual core
  • 2 discos duros de 1Tb replicados, para no perder ningún dato en caso de fallo de uno de ellos.
  • 2 tarjetas de ethernet
  • totalmente preconfigurado, y con el software más útil y necesario
  • con capacidad de instalar más software

El software que trae está comandado bajo un sistema operativo Arch Linux, por su buen manejo de arquitecturas ARM, y con el siguiente software listo para usar:

  • Para compartir archivos, calendario, contactos, etc, creando tu propia nube, no podía ser otro que ownCoud
  • Para crear tu web, blog, y publicar en la red WordPress, y compartir los contenidos mediante tus contactos de redes sociales.
  • Puedes gestionar tu colección multimedia mediante Mediagoblin y/o Trovebox
  • Crear tu propio nodo en una red descentralizada mediante idno
  • Para gestión de contenido tipo wiki Wang
  • Para los marcadores utiliza shaarli
  • Como gestor de RSS selfoss
  • Y como servidor de correo Mailpile

Como ves una buena cantidad de programas de software libre corriendo por sus circuitos!

Puedes leer más sobre el proyecto en su web:

  • http://indieboxproject.org/blog/

Y contribuir con ellos, ya que han puesto en marcha una campaña de crowdfunding para poder llevar a cabo este proyecto que desde luego tiene buena pinta. Para participar con tu apoyo en la campaña puedes hacerlo en este enlace:

  • https://www.indiegogo.com/projects/indie-box-let-s-bring-our-data-home
