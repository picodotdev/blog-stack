Estamos Promoviendo el uso de Linux y Software Libre correctamente?

Escrito por Jason Soto en la bitácora JsiTech, artículo original
Publicado el viernes, 07 de noviembre de 2014 a las 00:08 UTC
Etiquetado como software-libre, linux

Hace Días que quería crear esta entrada por la sencilla razón de que a veces muchos de los que queremos promover el Software Libre y el uso de Linux no lo hacemos de manera correcta, A que viene esto? Ultimamente he visto personas que quieren imponer el uso tanto del Software Libre como de Linux, ya sea porque entiende que el software propietario atenta contra la libertad o cualquier otra razón, pero debemos entender que todos vemos las cosas de manera distinta y a la mayoría de usuarios no le gusta que le impongan las cosas. Hace un tiempo recuerdo que vi en un grupo de Facebook de software libre, un Post de una persona que estaba solicitando ayuda o una aclaración con una implementación de un gestor base de datos, pero por el simple hecho de que lo estaba haciendo en Windows no le prestaron la ayuda, simplemente se limitaron a comentar que no le dan soporte a implementaciones hecha en productos Microsoft, cabe destacar que el gestor de base de datos funciona tanto en Windows como Linux. Entonces simplemente entendí que esa no era la manera, porque no se aprovecho la oportunidad para introducirle Linux al usuario explicándole sus bondades, y porque sería mejor implementarlo en ese sistema, entre otros.

Lo que deseo expresar aquí es que debemos aprovechar cualquier oportunidad para introducirle el Software Libre a los usuarios pero que ellos mismo entiendan los beneficios que les trae, no porque yo les diga, ese software no sirve, usa este y ya. Debemos darle a entender que el Software Libre es simplemente aquel que respeta nuestra libertad, La libertad de aprender y entender el software que estamos usando ya que nos permite estudiar el código sin las limitaciones que nos coloca el software propietario.

Muchos de los software propietarios que adquirimos no nos dan estos derechos ya que los que adquirimos es una licencia para utilizarlo pero controlando lo que podemos y no podemos hacer, y nunca llegamos a entender como funcionan estos Software y no conocemos lo que puede estar haciendo detrás de cámara, convirtiéndolo en una amenaza para nuestra privacidad. El sofware libre nos quita esas restricciones y esto nos da control sobre lo que hacemos con el software y como funciona. Y hablando de Seguridad, tenemos toda una comunidad trabajando todo el tiempo y no tenemos que esperar patch tuesday para resolver una brecha una vez ha sido dado a conocer.

No hablaré mucho sobre Software Libre, la idea es que tengamos un acercamiento distinto al momento de promover Linux y Software Libre, y que no nos vean como unos fanáticos Locos, que usualmente es lo que pasa y por esa razón la gente no lo adopta. En mi caso creo charlas en linea hablando sobre Software Libre y Linux, y realmente he logrado despertar el interés en las personas que lo ven y esto lo lleva a investigar un poco más sobre este tema y en la mayoría de los casos se convierten en usuarios de Linux y Software Libre.

Vamos a mostrarle a los usuarios que para cada Sofware Propietario hay una alternativa Libre y que respeta su libertad en todo momento.
