Hangout – Servidor Web Seguro en Linux con JackTheStripper

Escrito por Jason Soto en la bitácora JsiTech, artículo original
Publicado el sábado, 15 de noviembre de 2014 a las 00:31 UTC
Etiquetado como seguridad, jackthestripper, software-libre, linux

Todos los que siguen este blog han seguido el Desarrollo del deployer de Seguridad JackTheStripper, pero en algunos casos los usuarios no son muy dado a utilizar o correr algunas cosas si no lo han visto funcionar. Por tal razón he creado un hangout donde podrán ver en vivo de que se trata la herramienta y todo lo que hace paso a paso para asegurar un servidor Linux.

Además vamos a ver tambien como auditamos la seguridad de un servidor utilizando otra herramienta del que tambien he hablado en este blog llamada Lynis.

Participa en este Hangout donde conocerán el Deployer de Seguridad JackTheStripper, Un Proyecto iniciado por +Eugenia Bahit y al que nos hemos unido para llevar el soporte a otras Distros. Mas info en la web de Eugenia <www.eugeniabahit.com/proyectos/jackthestripper> y la Web de Jsitech <www.jsitech.com/jackthestripper>.

Que veremos?

– Qué es JackTheStripper
– Corriendo el Deployer de seguridad en un servidor con Linux CentOS
– Auditando el Nivel de Seguridad con Lynis

No se lo pierdan..

La cita es el Viernes 21 de Noviembre 2014 a las 8:00pm.
