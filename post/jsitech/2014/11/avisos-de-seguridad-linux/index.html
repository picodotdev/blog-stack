Avisos de Seguridad Linux

Escrito por Jason Soto en la bitácora JsiTech, artículo original
Publicado el lunes, 17 de noviembre de 2014 a las 13:50 UTC
Etiquetado como seguridad, linux

Tenia unos días sin publicar los avisos de seguridad por cuestiones de tiempo. Pero arrancamos nueva vez.

OpenSuse

Versiones Afectadas: 13.2

Actualización de Seguridad para OpenSSL para resolver múltiples vulnerabilidades. Actualizado a la versión 1.0.1j y resuelve:

  • SRTP Memory Leak (CVE-2014-3513)
  • Filtrado Memoria Sesiones Tickets (CVE-2014-3567)
  • Protección FallBack SSL 3.0 TLS_FALLBACK_SCSV (CVE-2014-3566)

Para actualizar

zypper in -t patch openSUSE-2014-671

Versión Afectada: 13.1

Actualización de Seguridad para libreoffice para resolver múltiples vulnerabilidades, estas son:

  • Vulnerabilidad Documento como Email
  • Vulnerabilidad Control Remoto Impress Use-after-free (CVE-2014-3693)

zypper in -t patch openSUSE-2014-661

Fedora

Versiones afectadas: 19

Actualización para el paquete python3. Resuelve Vulnerabilidad CVE-2014-4650: Módulo CGIHTTPServer no maneja correctamente los separadores de Rutas en las URL codificadas.

Para actualizar:

yum update python3

Versiones afectadas: 19,20

Actualización de Seguridad para aircrack-ng para resolver múltiples vulnerabilidades de seguridad, CVE-2014-8321, CVE-2014-8322, CVE-2014-8323, CVE-2014-8324

Para actualizar:

yum update aircrack-ng

Red Hat

Versiones afectadas: Todas

Actualización de Seguridad Crítica para flash-plugin. Resuelve múltiples Vulnerabilidades de seguridad en la forma que el plugin maneja cierto contenido SWF, permitiéndole a un atacante utilizar esta falla para crear contenido SWF modificados que causaría una falla en la aplicación o permitirle ejecutar código arbitrario. Se recomienda actualizar inmediatamente a la versión 11.2.202.418. (CVE-2014-0573, CVE-2014-0574, CVE-2014-0576, CVE-2014-0577,CVE-2014-0581, CVE-2014-0582, CVE-2014-0584, CVE-2014-0585, CVE-2014-0586,CVE-2014-0588, CVE-2014-0589, CVE-2014-0590, CVE-2014-8438, CVE-2014-8440,CVE-2014-8441)
