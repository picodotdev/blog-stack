Arachni – Framework Scanning de Seguridad Aplicaciones Web

Escrito por Jason Soto en la bitácora JsiTech, artículo original
Publicado el domingo, 02 de noviembre de 2014 a las 15:53 UTC
Etiquetado como ethical-hacking, seguridad, tutoriales, arachni, linux

Hoy me he topado con un proyecto bastante interesante en DarkNet que nos da una mano en el escaneo de una aplicación web. Como profesionales de Seguridad herramientas de este tipo nos pueden ayudar al momento de auditar aplicaciones y pórtales web, este proyecto es Arachni.

Arachni es un Framework desarrollado en Ruby de Código Libre, modular y lleno de Features que nos ayuda a los profesionales de Seguridad o administradores de Sistema evaluar el nivel de seguridad de nuestras aplicaciones web. Es inteligente por el hecho ya que se autoentrena monitoreando y aprendiendo del comportamiento de la aplicacion web durante el proceso de escaneo y es capaz de llevar a cabo meta-análisis usando ciertos factores para evaluar correctamente los resultados y evitar falsos positivos.

Arachni toma en consideración la naturaleza de la aplicacion web y puede detectar cualquier cambio mientras se mueve por las rutas de la aplicación y se puede ajustar en consecuencia de estos, es de esta manera que los vectores de ataque que pudieran ser indetectable se pueden ser manejados a la perfección.

En su nueva versión (1.0), dado por su ambiente de navegador integrado, puede auditar e inspeccionar código del lado del cliente, asi como contar con soporte para aplicaciones web de alta complejidad que hace uso extenso de JavaScript, HTML5, DOM y AJAX. Le permite además ser lo bastante versátil para cubrir cualquier caso de uso, desde el escaneo mediante la linea de comandos, una red distribuida de scanners de alto rendimiento, una librería Ruby para Scripts de auditoría, hasta una interfaz web multi-usuario y multi-escaneo.

Entre las características mas notables tenemos:

  • Múltiples opciones de Implementación
  • Ambiente de navegador integrado
  • Soporte para Pausa/Continuar
  • Soporte para Hibernación
  • Manejo de Sesiones Automática
  • Diversas opciones de alcance
  • Inteligencia, adaptación a cada aplicación web
  • Alto Rendimiento
  • Varios chequeos de Seguridad
  • Múltiples Plugins que aumentan su funcionalidad
  • Entre muchas otras,

Lo interesante de esto es que es bastante sencillo implementar esta herramienta ya que es un paquete autocontenido con todo lo que la aplicacion necesita para funcionar, solo debemos contar con Glibc 2.12 . Para descargarlo solo seguimos estos pasos.

Instalamos los prerequisitos

Para Centos/Fedora/RHEL

# yum install glibc

para Ubuntu/Debian

# apt-get install glibc

Descargamos la herramienta

#wget http://downloads.arachni-scanner.com/arachni-1.0.4-0.5.5-linux-x86_64.tar.gz

# tar xzvf arachni-1.0.4-0.5.5-linux-x86_64.tar.gz

# cd cd arachni-1.0.4-0.5.5

#cd bin

Una vez aqui, para realizar un escaneo por la linea de comandos

# ./arachni http://prueba.com

Para lanzar la aplicación web

# ./arachni_web -o ipservidor

Por defecto va a escuchar en el puerto 9292 pero pueden cambiarlo con la opción -p

para acceder

http://IPServidor:9292

usuario: admin@admin.admin

Pass: administrator
