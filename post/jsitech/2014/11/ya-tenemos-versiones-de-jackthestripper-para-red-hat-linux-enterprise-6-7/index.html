Ya Tenemos Versiones de JackTheStripper para Red Hat Linux Enterprise 6/7

Escrito por Jason Soto en la bitácora JsiTech, artículo original
Publicado el jueves, 13 de noviembre de 2014 a las 23:00 UTC
Etiquetado como seguridad, redhat, jackthestripper, software-libre, linux

Debo decir que he disfrutado mucho contribuir con el Desarrollo de JackTheStripper y es de satisfacción para mi cuando lo pruebo en otra distro y todo va bien. Ahora ya lo tenemos disponible para la Distro comercial Red Hat Linux Enterprise 6/7. Si usas esta Distro para implementar tus aplicaciones web críticas, pues te recomiendo que lo asegures con JackTheStripper que sigue las mejores prácticas de seguridad para este tipo de implementaciones.

Si aun no conoces que es JacktheStripper puedes ver de que se trata y lo que hace aqui.

Es bueno mencionar que RHEL requiere de una subscripción para tener acceso a los repos oficiales, pero para los que usan esta Distro entiendo que ya saben esto.

Para descargar y correr la herramienta solo deben seguir estos pasos como el usuario root.

RHEL 6

# wget http://www.jsitech.com/JTS_RHEL_6.tar.gz

# tar xzvf JTS_RHEL_6.tar.gz

# cd RHEL_Deployer

# ./dms.sh

RHEL 7

# wget http://www.jsitech.com/JTS_RHEL_7.tar.gz

# tar xzvf JTS_RHEL_7.tar.gz

# cd RHEL_Deployer

# sh Prerun.sh

# ./dms.sh

Sigan las instrucciones del Deployer
