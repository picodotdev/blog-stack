Bitnami, Una tienda de Aplicaciones para Servidores

Escrito por Jason Soto en la bitácora JsiTech, artículo original
Publicado el martes, 16 de septiembre de 2014 a las 12:33 UTC
Etiquetado como bitnami, linux, stack, noticias, software-libre, generales

Como siempre andando en la red encontre otro proyecto que me pareció bastante interesante y enfocado a aquellos usuarios sin muchos conocimientos de Administración de sistemas Linux ni en la implementación de muchas aplicaciones. Para ponerles un ejemplo, un usuario que quiere montar WordPress en un servidor pero no conoce muchos de los pasos a seguir que incluye, Descargar el paquete de WordPress, instalar los prerequisitos, la base de datos, etc, esto le puede llevar mucho tiempo y es aquí que este proyecto llamado Bitnami entra en acción.

Bitnami es una tienda de aplicaciones para servidores, es una librería con las más populares aplicaciones para servidores y ambientes de desarrollo que pueden ser instalado con un solo click o ejecutando un comando. Bitnami se encarga de compilar y configurar las aplicaciones y toda sus dependencia como base de datos y demás, para que funcione inmediatamente. El resultado es un solo paquete que luego es distribuido como instalador nativo, máquina virtual o imágenes para la nube. Según el portal estos paquetes proveen una solución consistente, segura y optimizada al momento de implementar una aplicación en cualquier plataforma.

Lo mejor de esta solución es que le reduce el tiempo de implementación de la aplicación al usuario, al este no tener que invertir tiempo en investigar como instalar la aplicación.

Características

  • Las aplicaciones son autónomas y empaquetadas con todas las librerias y base de datos necesarias para ejecutarse
  • Las actualizaciones están disponibles poco tiempo despues de ser lanzadas para asegurar que las aplicaciones cuenten con un excelente nivel de seguridad.
  • La comunidad elije que aplicaciones son empaquetadas
  • Según Bitnami, cada vez que se da a conocer una vulnerabilidad o falla de seguridad, las aplicaciones son actualizadas en el menor tiempo posible
  • Las aplicaciones son configuradas para funcionar en los escenarios mas comunes
  • Las aplicaciones son empaquetadas siguendo un estándar consistente, asi que la experiencia de implementarlas de forma nativa, en una máquina virtual o en un hosting es casi idéntica, asi que es fácil moverse de desarrollo a producción.

Que plataformas soporta?

Bitnami soporta las siguentes plataformas

Sistemas operativos: Todas las distros de Linux, OSX, Windows

Virtualización: VMWare, VirtualBox, Xen, KVM, Parallels.

Nube: Amazon Web Services, Microsoft Azure, VMWare vCloud, OpenStack, CloudStack.

Todos los paquetes de aplicaciones de Binami son Open Source y pueden ser libremente utilizado y redistribuido.

Algunos de los Paquetes de aplicaciones con la que cuenta Bitnami incluyen: WordPress, Joomla, Drupal, Nginx, Moodle, ownCloud, EspoCRM, Magento, GitLab, SugarCRM, y muchísimas aplicaciones mas, y se seguirán agregando. Para ver todo el listado de aplicaciones pueden acceder aqui.

Me hubiese gustado toparme con esta solución hace unos días que estaba implementado un servidor de control de versiones con GitLab para los desarrolladores, debo comentarle que me tomó mucho tiempo ya que siempre tenía errores con algunas librerías faltante, la instalación de Ruby arrojaba errores, mas todo el tiempo que me tome para implementar las otras dependencias. En total fue casi todo un día de trabajo. Tome el instalador de GitLab de Bitnami para probar y la instalación tomo solo algunos minutos, y solo fue acceder via web a la interfaz, tan sencillo como eso.
