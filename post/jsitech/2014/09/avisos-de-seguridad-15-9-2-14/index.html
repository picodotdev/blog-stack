Avisos de Seguridad 15/09/2014

Escrito por Jason Soto en la bitácora JsiTech, artículo original
Publicado el lunes, 15 de septiembre de 2014 a las 19:05 UTC
Etiquetado como linux, noticias, avisos, seguridad

Fedora

Actualización de Seguridad para:

curl Fedora 20, Soluciona 2 vulnerabilidades en el manejo de los Cookies.

GraphicsMagick Fedora 19, Soluciona vulnerabilidad que permite ejecución de Código remoto.

LibreOffice Fedora 20, Soluciona Vulnerabilidad que le permite a un atacante enviar un documento que genera un aviso de actualizar los links, que aunque sea cancelado es ejecutado y el código malicioso inserta en el documento una imagen OLE2 en el sistema de archivo de la victima, La exposición de los datos es posible si el documento es distribuido a terceros.

Procmail Fedora 20, Soluciona vulnerabilidad que permite ejecución de Código remoto

Actualizar con yum

OpenSuse

Actualización de Seguridad para LibreOffice, Soluciona vulnerabilidad de corrupción de memoria y exposición de datos usando objetos especialmente diseñado.

Para aplicar los Parches, ejecutar:

-OpenSuse 12.3/13.1

# zypper in -t patch openSUSE-2014-540

Ubuntu

Actualización de Seguridad para Curl, Soluciona Vulnerabilidades en el manejo de los cookies. Se recomienda actualizar. Versiones afectadas 14.04/12.04/10.04 LTS

Red Hat

Actualización de Seguridad para:

axis, Soluciona vulnerabilidad Bypass verificación de Hostname

python-django-horizon, Soluciona múltiples Vulnerabilidades de Seguridad

qemu-kvm-rhev, Soluciona vulnerabilidades que permite ejecucion de código Remoto.

Para aplicar los parches, seguir las indicaciones de https://access.redhat.com/articles/11258

Mageia

Actualización de Seguridad para:

dump, Soluciona vulnerabilidades que pueden permitir un ataque de denegación de servicios, y ejecución de código remoto.

Glibc, Soluciona dos vulnerabilidades

libgadu, Soluciona problema de no verificación de Certificados SSL

mariadb, Soluciona vulnerabilidad que permite ejecutar código remoto usando los archivos temporales de MyISAM.

Moodle, Soluciona dos vulnerabilidades de Seguridad.

Suse

Actualizacion de Seguridad para:

firefox, Soluciona múltiples vulnerabilidades de Seguridad que permiten la ejecución de código arbitrario.

Flash player, Soluciona Múltiples Vulnerabilidades que pueden permitir la ejecución de código arbitrario y otros ataques.

Glibc, Solucion vulnerabilidad que permite la elevación de privilegios en el sistema víctima.

Se recomienda actualizar el sistema.
