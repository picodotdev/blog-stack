JackTheStripper para Fedora Linux 19/20

Escrito por Jason Soto en la bitácora JsiTech, artículo original
Publicado el lunes, 27 de octubre de 2014 a las 00:14 UTC
Etiquetado como noticias, jackthestripper, fedora, software-libre, linux

Seguimos desarrollando activamente la herramienta de Hardening para Linux, JackTheStripper,  y en esta venimos con versiones para Fedora Linux 19 y 20. Aunque pienso que esta Distro está más enfocada al Desktop, si es utilizada como server en muchas implementaciones, así que decidimos lanzar la herramienta para esta distro por igual.

Recapitulando un poco JackTheStripper es una herramienta Software libre que esta enfocado a aquellos programadores que no tienen muchos conocimientos de Administración de Sistemas Linux, ni como asegurar esos servidores donde van a implementar su sistema. Lo que le permite es configurar un servidor web de manera rápida sin preocuparse de la optimización o la Seguridad que es un tema muy importante al momento de publicar una aplicación para internet.

Esta versión para Fedora se encarga de:

  • Configurar un hostname
  • Reconfigurar la zona horaria del sistema
  • Actualizar el sistema operativo por completo, asegurando contar con las últimas versiones disponibles de cada uno de los paquetes del sistema
  • Crear un nuevo usuario con privilegios de administrador, para que pueda manejar su sistema de forma segura, sin depender de conexiones remotas mediante el usuario root
  • Generar las llaves públicas RSA necesarias, para que el acceso SSH a su servidor, pueda efectuarse solo y únicamente desde su ordenador
  • Configurar, optimizar y asegurar el servicio SSH, haciéndolo inmune frente a intentos de acceso no autorizados
  • Configurar el Firewall, generando reglas de seguridad estrictas en iptables que liberen de riesgos de accesos no autorizados a su servidor
  • Reforzar la seguridad contra ataques por fuerza bruta, mediante la instalación, configuración y optimización de fail2ban
  • Hardening a nivel de Kernel, configurando Sysctl
  • Instalar, configurar y optimizar MariaDB
  • Instalar, configurar y optimizar PHP 5, para que la ejecución de sus aplicaciones sea tan segura como eficaz en el rendimiento.
  • Reforzar la seguridad de Apache, mediante la instalación, configuración y optimización de módulos como ModSecurity y ModEvasive
  • Instalar paquetes adicionales que pueden ser elementales tanto para la gestión del sistema como para el deploy de sus propias aplicaciones.
  • Configurar Nano, Vim
  • Agregar tareas de actualización y mantenimiento periódicas al Cron, para evitar que usted deba preocuparse por ellas

Para descargar y utilizar la Herramienta,

Fedora 19

# wget http://www.jsitech.com/JTS_Fedora_19.tar.gz
# tar xzvf JTS_Fedora_19.tar.gz
# cd Fedora_Deployer/

# sh Prerun.sh
# ./dms.sh

Fedora 20

# wget http://www.jsitech.com/JTS_Fedora_20.tar.gz
# tar xzvf JTS_Fedora_20.tar.gz
# cd Fedora_Deployer/

# sh Prerun.sh
# ./dms.sh

NOTA: El Deploy debe hacerse sobre una instalación limpia de Fedora Linux 19/20

En caso de presentar cualquier inconveniente con la herramienta favor repórtenlo a https://bugs.launchpad.net/jackthestripper
