Tarjeta Wireless Broadcom BCM43000 en Kali Linux

Escrito por Jason Soto en la bitácora JsiTech, artículo original
Publicado el martes, 13 de enero de 2015 a las 17:08 UTC
Etiquetado como ethical-hacking, seguridad, kali, broadcom, tutoriales, linux

Recientemente instale Kali Linux en una Laptop HP que tiene una tarjeta Wireless Broadcom, tengo un Dual Boot con Ubuntu donde funciona sin problemas pero en Kali no fue asi. Estuve andando por google intentando todas las posibles soluciones sin éxito y asi mismo vi un sin número de usuarios que tenían el mismo caso y hacían las preguntas en Foro sin tener éxito. Luego de un rato de búsqueda me he topado con un script que funciona muy bien y con el que pude instalar los drivers con exito. Aquí se lo comparto si como yo tienen la misma experienca.

Primero validemos que tenemos una tarjeta wireless Broadcam

# lspci -vnn |grep Network

00:19.0 Ethernet controller [0200]: Intel Corporation 82579V Gigabit Network Connection [8086:1503] (rev 04)

24:00.0 Network controller [0280]: Broadcom Corporation BCM43228 802.11a/b/g/n [14e4:4359]

En mi caso tengo una BCM43228, aunque el script fue creado para la 43142 funciona sin problemas y entiendo que debe funcionar en cualquier otra.

Vamos ahora a descargar el script y a correrlo

# wget https://gist.githubusercontent.com/chokepoint/11305606/raw/52109f648c2ad5ec6a65bf8eee78fbda15fa1cfb/bcm43142_drivers.sh

# chmod +x bcm43142_drivers.sh

# ./bcm43142_drivers.sh

Aqui el contenido del script.

#!/bin/bash

# chmod +x broadcom_drivers.sh

# ./broadcom_drivers.sh

mkdir broadcom

cd broadcom

apt-get install linux-headers-`uname -r` build-essential -y

if [ `uname -m` == “i686″ ] ; then

wget http://www.broadcom.com/docs/linux_sta/hybrid-v35-nodebug-pcoem-6_30_223_141.tar.gz

else

wget http://www.broadcom.com/docs/linux_sta/hybrid-v35_64-nodebug-pcoem-6_30_223_141.tar.gz

fi

tar -xvf hybrid-v35*

wget http://www.mindwerks.net/wp-content/uploads/2013/10/wl_3.10.patch

patch -p2 < wl_3.10.patch

make

cp wl.ko /lib/modules/`uname -r`/kernel/net/wireless/

depmod

rmmod bcma

echo “blacklist bcma” >> /etc/modprobe.d/blacklist.conf

modprobe wl
