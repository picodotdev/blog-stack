Que es y como instalar Wine mediante PPA en Ubuntu

Escrito por Liher en la bitácora El blog de Liher, artículo original
Publicado el jueves, 10 de julio de 2014 a las 16:36 UTC
Actualizado el jueves, 10 de julio de 2014 a las 16:36 UTC
Etiquetado como ubuntu, ppa, instalar, wine

En esta ocasión os voy a hablar de una aplicación que podemos instalar en Ubuntu y que puede ser de mucha utilidad para aquellos usuarios que provienen de Windows y todavía necesiten usar algunas aplicaciones de ese sistema operativo. El programa en cuestión se llama Wine, os voy a hablar un poco de el y después os mostrare como instalar su ultima versión disponible mediante PPA en Ubuntu, tengo que decir que este PPA sirve para todas las versiones de Ubuntu, incluso ya funciona para Ubuntu 14.10.

Si queremos usar alguna aplicación de Windows en Ubuntu tenemos algunas opciones, hay algunos programas que tienen su versión oficial para Windows y Linux, pero otros no. Cuando nos encontramos en la situación en que no tenemos una versión oficial del programa deseado de Windows para Ubuntu podemos hacer dos cosas, una es crearnos una maquina virtual e instalar Windows en ella y la otra es usar Wine u otros programas similares. Wine nos proporciona un entorno muy parecido al de Windows en Ubuntu, con librerías y demás, también nos “crea” una unidad “C:” donde podemos navegar por los archivos que tengamos instalados mediante Wine.

 

 

Wine no soporta todos los programas de Windows pero el numero va creciendo versión tras versión y ahora mismo no sabría decir cuantos son, pero son muchos. Con Wine podemos instalar dichos programas o juegos y así podemos usarlos en Ubuntu en la mayoría de los casos.

Actualmente va por la versión 1.7.21, esta es la versión en desarrollo, pero la ultima versión estable es la 1.6.2. Aquí os dejo un enlace a la pagina oficial de Wine, esta en español y allí podéis encontrar bastante información acerca de esta buena utilidad:

  • Pagina oficial de Wine

En ella tenéis la opción de descargaros el propio Wine, también podemos instalarlo desde los repositorios oficiales de Ubuntu, pero si queremos tener este programa actualizado, lo cual es aconsejable para que cada vez de saquen una nueva versión se actualice. Si queremos instalarlo añadiendo el PPA correspondiente solo tenemos que abrir una Terminal de nuestro Ubuntu y escribir el siguiente comando:

sudo add-apt-repository ppa:ubuntu-wine/ppa

Luego tenemos que recargar la paqueteria para que reconozca la nueva versión con el siguiente comando:

sudo apt-get update

Y finalmente instalar Wine con el siguiente comando:

sudo apt-get install wine1.7

Es un programa bastante pesado, ocupa bastante espacio, alrededor de unas 500 MB mas o menos. Después de instalarlo ya podremos ejecutar aplicaciones Windows en nuestro Ubuntu gracias a el.

¿Lo conocíais? Yo antes lo usaba, pero con el tiempo he ido encontrando aplicaciones para substituir a las de Windows hasta el punto de no necesitar ninguna, salvo el Photoshop en ocasiones muy puntuales. Ahora ya no lo tengo ni instalado en mi ordenador. Me he decantado por tener una maquina virtual con Windows 7 para el Photoshop y para alguna partida esporádica al Age of Empires II.

Un saludo.
