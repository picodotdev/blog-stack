Servidor web virtual con Xubuntu 14.04 disponible para descargar

Escrito por Liher en la bitácora El blog de Liher, artículo original
Actualizado el martes, 27 de mayo de 2014 a las 11:14 UTC
Etiquetado como xubuntu-14-4, descargar, ubuntu, servidor-web, servidor

Para todos aquellos que quieran el servidor web virtual hecho con Xubuntu 14.04 y que no tengan ganas de hacerlo siguiendo los tutos que he hecho aquí os voy a dejar el enlace de descarga para que lo podáis tener. Lo he subido a Google Drive, pero antes de poner el enlace para que lo podáis descargar me gustaría hacer un pequeño resumen de lo que tiene, poneros los datos necesarios para usarlo y algunos consejos.

Este servidor web tiene los siguientes características:

  • Esta hecho con Virtual Box y con el Extension pack previamente instalado.
  • Tiene activadas y funcionando las Guest Additions.
  • Le he instalado el servidor web Apache
  • También tiene instalado Mysql, PHPMyadmin y PHP
  • Tiene servicio de FTP
  • En el escritorio tenéis la ultima versión de WordPress y Joomla, ambos en español.
  • He instalado el Geany, un editor de texto orientado a la programación, yo lo uso para hacer webs
  • Tiene creados lanzadores para una rápido acceso a la Terminal, Geany, administrador de archivos y Firefox.
  • Todo esta configurado y funcionando correctamente.

El archivo con el servidor web virtual, ocupa 1,7 GB aproximadamente. Cuando lo descargueis solo teneis que hacer doble click sobre el y luego en el boton que vereis con el texto “Importar”. La configuracion de red que tiene incluida es la que yo uso y puede ser que no os sirva a vosotros, para ello hay que seguir los pasos de este articulo:

  • Paso 5 de la creación del servidor web virtual

No tiene un directorio compartido configurado, si queréis usar uno podéis ver como crearlo en el siguiente articulo:

  • Paso 6 de la creación del servidor web virtual

Si tenéis algún problema con la resolución de la pantalla o con la carpeta compartida, puede que tengáis que volver a reinstalar las GA, en el siguiente articulo tenéis como hacerlo:

  • Paso 4 de la creación del servidor web virtual

En un principio no debería dar problemas, yo he usado este servidor web virtual con Xubuntu 14.04 en tres ordenadores diferentes y sin problemas. El servidor tiene dos usuarios:

  • root : su contraseña es “contraseña”
  • usuario : su contraseña es “contraseña”

Con esos datos tendréis acceso a todo, lo he configurado así para que sea mas fácil de usar y no tengamos que estar pensando que contraseña es para cada cosa, lo he hecho también pensando en los olvidadizos como yo

Por el momento no recuerdo nada mas, si tenéis alguna duda o consulta dejadla aquí en forma de comentario para que si otro usuario tiene la misma que pueda encontrar aquí su respuesta rápidamente. Sin mas os dejo el enlace de descarga y espero que os sirva:

  • Descargar servidor web virtual Xubuntu 14.04
