Como instalar Spotify en Ubuntu 14.04

Escrito por Liher en la bitácora El blog de Liher, artículo original
Actualizado el lunes, 14 de julio de 2014 a las 19:00 UTC
Etiquetado como ubuntu, instalar, instalar-spotify, spotify-ubuntu, spotify

Para todos aquellos que todavía no conozcan este programa, el Spotify, os voy a hablar un poco de el y de sus bondades, ademas os voy a mostrar como instalarlo en Ubuntu 12.04 y Ubuntu 14.04, para ambas versiones el proceso es el mismo. Yo lo conoci hace ya un tiempo y, aunque me gusta mucho, no lo uso habitualmente porque, según lo que este haciendo, la música no me deja concentrarme.

 

 

 

 

Spotify es un programa multiplataforma, esto quiere decir que lo podemos usar en Windows, Linux y MAC, que sirve básicamente para escuchar música a través de Internet. La cantidad de canciones es interminable, hay de todo y mucho. Podemos escuchar canciones sueltas, o discos enteros. Ademas tienes la opción de estar al día escuchando las novedades en las distintas emisoras disponibles. Siempre que tengas conexión a Internet podrás escuchar cualquier disco o canción, sea del artista que sea.

Después de instalar Spotify podemos configurar listas de canciones, escuchar emisoras de radio, incluso puedes personalizar tu propia emisora según tus preferencias. La verdad es que el manejo del programa es bastante sencillo e intuitivo, podréis comprobarlo después de instalar. Si queréis instalar este magnifico programa en Ubuntu aquí os dejo los comandos necesarios para ello, los he probado en Ubuntu 12.04 y 14.04 y en los dos ha funcionado todo correcto. Lo primero que tenemos que hacer es abrir la Terminal y teclear lo siguiente:

sudo gedit /etc/apt/sources.list

Después de teclear vuestra contraseña se abrirá un editor de texto con un archivo, tenéis que ir hasta el final del archivo y escribir en una linea nueva lo siguiente:

deb http://repository.spotify.com stable non-free

Después de esto guardáis el archivo y cerráis el Gedit. Ahora en la Terminal vamos a añadir la clave publica con el siguiente comando:

sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 94558F59

Finalmente recargamos los paquetes con:

sudo apt-get update

Y ahora vamos a instalar Spotify en Ubuntu con este comando:

sudo apt-get install spotify-client

El Spotify es un programa un poco pesado, ocupa algo mas de 100MB, una vez terminado de instalar ya podremos escuchar cualquier canción en Ubuntu gracias a el.
