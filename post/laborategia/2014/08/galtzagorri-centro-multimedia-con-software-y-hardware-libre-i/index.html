Galtzagorri; centro multimedia con software y hardware libre (I)

Escrito por Dabid Martinez en la bitácora Laborategia, artículo original
Actualizado el viernes, 05 de septiembre de 2014 a las 10:12 UTC
Etiquetado como raspberry-pi, aplicaciones, cosas-que-hago, blog-stack, makers, xbmc, en-castellano, tailerra, software-libre, openelec, linux

Una de geekismo personal

El objetivo de este proyecto es montar un centro multimedia casero y de bajo coste que me permita disfrutar mis series, películas y música favorita desde la comodidad del sillón de mi casa. Para ello emplearé diferentes software y hardware, en este caso todo libre. La idea inicial es utilizar para este proyecto una Raspberry Pi que tengo muerta de risa en el armario, el software XBMC para que trabaje de media center, Yatse como mando a distancia para el media center y la LG que vive en la sala.

Una vez terminado el nuevo centro de entretenimiento se llamará; Galtzagorri, por lo jugeton del personaje mitológico vasco.

Para el software de gestión del media center podría utilizar una Debian como base y después ir instalando los paquetes, pero las compañeras de OpenELEC (Open Embedded Linux Entertainment Center) ya han hecho el trabajo mas tedioso. Han desarrollado un completo sistema operativo base Linux para la placa Raspberry Pi sobre el que han pre-instalado XBMC y parametrizado para poder usarlo sin tener ningún conocimiento avanzado de informática. Me decanto por usar este proyecto.

Es mas simple de lo que muchos podéis pensar, así que manos a los bit. Cogemos una tarjeta SD, en mi caso he usado una de 8 GB lo cual me da mas que de sobra para la instalación del sistema operativo y el software, pero es la que mas tenía a mano, seguro que una mas pequeña valdría ya que OpenELEC pesa sobre los 140 MB. La formateamos para poder comenzar a usar el instalador de OpenELEC (me gusta dejar limpias las cosas que voy a utilizar, para no tener demasiados problemas). En lo que el PC formatea la tarjeta SD podemos ir descargando la última versión de OpenELEC para la Raspberry Pi, que encontraremos de forma completamente gratuita (donad malditos) en la página principal del proyecto.

Una vez descargado simplemente tendremos que descomprimir el fichero dentro de cualquier directorio, acceder al directorio descomprimido a través de la terminal y ejecutar el siguiente comando:

sudo ./create_sdcard /dev/mmcblk0

Donde /dev/mmcblk0 es el dispositivo (que no partición) sobre el que queremos instalar OpenELEC, puede que en tu caso sea otro, sabrás identificar cual es usando el comando:

dmesg | tail

La última linea de respuesta de ese comando debería darte el nombre del último dispositivo conectado a tu Linux, en nuestro caso la tarjeta SD.

Le damos el beneplácito de esperar unos minutos, ya que una vez terminado el proceso de copia no tendremos mas que insertar la SD dentro de la Raspberry Pi, alimentar esta con su cargador de corriente, conectarle un cable HDMI y el otro extremo a nuestro televisor, ya que como podréis ver el sistema OpenELEC funciona perfectamente.

Una vez conectada simplemente tendremos que seguir el asistente para la configuración del nuevo media center, en 5 sencillos pasos podremos tener todo listo para disfrutar la máximo de el. Aquí os dejo unas fotos (no he conseguido hacer capturas) del proceso de configuración:

  1. Al encender el media center nos da la bienvenida y tendremos que seleccionar el idioma en el que queremos trabajar con el, a los amigos vascos les recuerdo que el euskera es un idioma que viene por defecto y que esta completamente traducido.
  2. El segundo paso sera ponerle un nombre a nuestro dispositivo, como ya he dicho al inicio del post el mio sera bautizado como Galtzagorri.
  3. Tendremos que configurar la red, en mi caso voy a usar una red cableada, el mismo XBMC detecta todo lo necesario para que el media center disponga de acceso a Internet. Si no disponéis de cable al lado de vuestra tele podéis conectar una tarjeta de red inalambrica por USB.
  4. Para el acceso a dispositivo me he decantado por activar SSH y Samba, SSH por que siempre utilizo Linux y Samba “por si acaso”.
  5. Una vez hayamos pasado todos los pasos de la configuración de forma correcta nos dará las gracias y ya lo tendremos completamente funcional

En unos días os explicaré cómo tenemos que hacer para la configuración avanzada del sistema así como para la configuración del mando a distancia; para lo cual usaremos nuestro smartphone o tablet Android, así nunca perderemos el mando
