Metadata Anonymisation Toolkit; limpieza de metadatos en los confines de tu HD

Escrito por Dabid Martinez en la bitácora Laborategia, artículo original
Actualizado el martes, 27 de mayo de 2014 a las 13:20 UTC
Etiquetado como blog-stack, internet, activismo-social, en-castellano, software-libre, comunicacion, hackinbadakigu, privacidad, cosas-que-veo, seguridad

Seguro que pocas veces te has preguntado si se puede sacar información sobre ti de una foto al subirla a flickr, o de un odt al pasarlos por email, de una grabación de una radio libre… pero algunos te diremos que SI, y es que, como bien nos hizo saber Diario Turing los metadatos de nuestros ficheros dicen DEMASIADO sobre nosotros.

Pero comencemos por el principio, o como dijo Jack el destripador, vamos por partes… ¿que es un metadato?

La palabra metadata (o metadatos) se refiere a “los datos de los datos”. Por ejemplo, en el caso de un email o una llamada, son metadatos el horario, la fecha en que se envió y la localización desde la que se conectó el usuario la última vez, entre otros.Diario Turing

En términos de privacidad, los metadatos de nuestros ficheros contienen un exceso de información sobre nosotros que no tiene por que ser publica, a no ser que seamos conscientes de que queremos hacerla publica.

MAT, o Metadata Anonymisation Toolkit, es un SET de herramientas que podemos usar por linea de comandos o a través de una interfaz de usuario que nos permite eliminar los metadatos de un fichero para poder pasarlo “en limpio” allá donde queramos enviarlo. Esta desarrollado en Python por lo que podemos usar en cualquier distribución de GNU/Linux.

El proceso de instalación es muy simple, en primer lugar resolveremos las dependencias:

1

sudo apt-get install python-hachoir-core python-hachoir-parser python-pdfrw python-cairo python-poppler python-mutagen libimage-exiftool-perl libimage-info-perl

Posteriormente clonaremos el repositorio del GIT oficial de la plataforma:

1

git clone git://gitweb.torproject.org/user/jvoisin/mat.git

Para finalizar con el proceso de instalación daremos permisos de ejecución a la interfaz de usuario desde el directorio raíz del repositorio clonado:

1

sudo chmod  +x mat-gui

Finalizado el proceso de instalación pasaremos a ejecutar la interfaz de usuario de MAT:

1

python mat-gui

Esto nos permitirá ejecutar el programa y comenzar a usarlo, su uso es muy simple, se compone de un botón para añadir el documento que queremos limpiar, un botón para limpiar los metadatos de los archivos y poco mas, ya que realmente poco mas necesitamos.

Una vez hayamos limpiado el documento el software nos recreara una copia del mismo añadiéndole el sufijo -cleaned a nombre del fichero original, ese sera, y no otro, el fichero que deberemos enviar a nuestros contacto.

A ver si con esto conseguimos tener un poco mas de libertad individual y colectiva, que al fin y al cabo, es de lo que se trata. Nosotros por nuestra parte trataremos de incluir este programa a la lista del taller de seguridad y privacidad en Internet
