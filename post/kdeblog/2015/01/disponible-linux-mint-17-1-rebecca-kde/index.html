Disponible Linux Mint 17.1 Rebecca KDE

Escrito por baltolkien en la bitácora KDE Blog, artículo original
Publicado el sábado, 10 de enero de 2015 a las 08:10 UTC
Etiquetado como distribuciones, linux-mint

Ya está con nosotros, una de las distribuciones más populares del mundo se actualiza. Desde el pasado 8 de enero ya tenemos disponible Linux Mint 17.1 Rebecca KDE. Veamos sus novedades.

Disponible Linux Mint 17.1 Rebecca KDE RC

Ya tenemos entre nosostros la  versión definitiva de Linux Mint 17.1 “Rebecca” para KDE, una distribución que ya está disponible con otros escritorios como Cinnamon o MATE, pero que siempre tarda un poco más en llegar con el escritorio KDE.

 

Como podemos leer en The Linux Mint Blog:

El equipo de Linux Mint se complace en anunciar Linux Mint 17.1 Rebecca KDE.

Esta versión de Linux Mint 17.1 es una versión LTS (long term support) que será mantenida hasta el 2019. Viene con software actualizado y trae muchas mejoras y nuevas características para hacer su escritorio aún más cómodo de usar.

Novedades

De esta forma nos ofrece algunas novedades como:

  • KDE 4.14
  • Linux kernel 3.13
  • Integración con KWallet
  • Update Manager
  • Mejoras en la ventana de Login
  • Mejoras en el sistema como en el comando pastebin o search
  • Mejoras visuales como una extensa colección de fondos de pantalla.
  • Otras mejoras como con USB Image Writer o el Gestor de Software

Los requisitos mínimos de Linux Mint 17.1 son los siguientes:

  • x86 processor (Linux Mint 64-bit requiere un procesador de 64-bit. Linux Mint 32-bit funciona con los dos tipos de procesadores 32-bit y 64-bit).
  • 2GB RAM
  • 10 GB de disco duro (recomendado 20GB).
  • Tarjeta gráfica de resolución mínima de 1024×768
  • Lector de DVD o puerto USB

 Instrucciones para su actualización:

  • Si desea actualizar desde Linux Mint 17.1 RC, sólo tiene que iniciar el Gestor de actualizaciones e instalar cualquier actualización Nivel 1 disponible.
  • Si desea actualizar desde Linux Mint 17, por favor, espera unos días mientras se prepara una nueva versión del Gestor de actualizaciones.
