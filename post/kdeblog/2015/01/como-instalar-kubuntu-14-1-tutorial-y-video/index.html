Cómo instalar Kubuntu 14.10: tutorial y vídeo

Escrito por baltolkien en la bitácora KDE Blog, artículo original
Publicado el jueves, 29 de enero de 2015 a las 08:00 UTC
Etiquetado como como, guia, video, tutorial, kubuntu

Ya ha pasado a mejor vida el tiempo es que instalar un sistema operativo GNU/Linux era una odisea. No obstante nunca está de más hacer pequeños tutoriales para que los recién llegados no teman dar el salto. Bienvenidos a cómo instalar Kubuntu 14.10 en x sencillos y cómodos pasos.

Cómo instalar Kubuntu 14.10

Actualmente el proceso de instalación de casi cualquier sistema GNU/Linux se ha simplificado mucho, además la detección de hardware ha mejorado ostensiblemente haciendo que en la mayoría de las ocasiones todo queda perfectamente configurado una vez finalizada la instalación.

Aún así nunca está de más realizar una explicación detallada del proceso para aquellas personas que se introducen en el mundo de las sistemas operativos GNU/Linux, como es el caso del siguiente tutorial de cómo instalar Kubuntu 14.10.

El proceso se inicia bajando la imagen de la distribución, comunmente llamada iso, de la página oficial de Kubuntu. En nuestro caso hemos bajado la versión Kubuntu 14.10 de 32 bits de la página de descargas.

Una vez bajada se graba en un DVD. Esto se puede hacer mediante Nero para Windows siguiendo los pasos de este vídeo. Para sistemas GNU/Linux podemos utilizar K3b o Brasero (también os dejo un vídeo)

Ahora toca arrancar el ordenador desde el DVD, para ello debéis activarlo en la Bios: ponéis el disco en vuestro lector y encendéis  el ordenador.

De esta forma el ordenador iniciará el programa de instalación de Kubuntu 14.10 de forma automática, debiendo aparecer las pantallas de arranque.

Ahora empiezan las pocas elecciones que nos permite el instalador. Empezamos eligiendo el idioma de la instalación y decidiendo si simplemente queremos probar Kubuntu, recomendado para probar si todo funciona bien con tu equipo antes de proceder a la instalación, o si realmente queremos instalarlo, que será nuestro caso.

A continuación nos aparece una pantalla de requisitos mínimos y la posibilidad de activar la red, incluso la wifi, o si queremos software de terceros. En nuestro caso, simplemente seleccionamos las opciones básicas.

Ahora toca  preparar el disco duro, como es una instalación sencilla dejamos las opciones por defecto.

La siguiente pantalla es la configuración de la zona horaria

Y la siguiente el tipo de teclado.

A continuación toca definir un usuario donde rellenaremos nombre, nick y contraseña (dos veces), el resto lo dejamos igual.

 

Una vez aceptado empezará la instalación propiamente dicha, que durará más o menos según el tipo de ordenador. Es el momento de tomarse un café.

Tras reiniciar el ordenador, no sin antes quitar el DVD de instalación, ya tendremos instalado Kubuntu 14.10.
