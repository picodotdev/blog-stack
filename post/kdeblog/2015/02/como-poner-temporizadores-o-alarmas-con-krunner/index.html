La ventaja de los sistemas libres como el escritorio Plasma de KDE es que cualquier desarrollador puede implementar mejoras de forma sencilla. Así es como aparecen proyectos como Timer and Alarm for KRunner o cómo poner temporizadores o alarmas con Krunner.

Cómo poner temporizadores o alarmas con Krunner

En la categoría de Mejoras para KDE de KDE-Apps y de la mano de Konstantin Enchant nos encontramos un excelente complemento para KRunner, el lanzador de aplicaciones vitaminado de KDE.

Se trata de Timer and Alarm for KRunner, una pequeña aplicación que aumenta las no pocas funcionalidades de KRunner añadiendo la posibilidad de poner temporizadores y alarmas directamente sobre KRunner.

Timer and alarm for KRunner soporta múltiples temporizadores e infinitas alarmas y su funcionamiento es bastante intuitivo ya que basta escribir con unos comandos qué queremos realizar.

A modo de ejemplo, para poner un temporizador basta escribir: timer 5m feed de cat, como se ve en la imagen inferior que traducido sería algo así como “Pon un temporizador para dentro de 5 minutos y que me avise de dar d comer al gato”

O el siguiente ejemplo que sirve para poner una alarma a una hora determinada para ver las noticias.

¿Cómo instalar Timer and Alarm for KRunner?

Para instalar esta aplicación tienes un par de posibilidades. La primera es la clásica de descargar e instalar. Concretamente:

1. Descargar krunner-alarmtimer.zip
2. Ejecutar en una consola: $ plasmapkg –type runner -i krunner-alarmtimer.zip

La segunda es mediane la clonación y la instalación desde el código fuente. Paso a paso se realiza de la siguiente forma, abrimos una consola y escribimos:

git clone git@github.com:sirkonst/krunner-timer.git
cd krunner-timer/
make install

Ahora simplemente debemos ejecutar KRunner e ir escribiendo frases como las que hemos visto en las imágenes anteriores.
