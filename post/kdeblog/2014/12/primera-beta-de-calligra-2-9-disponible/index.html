Primera beta de Calligra 2.9 disponible

Escrito por baltolkien en la bitácora KDE Blog, artículo original
Publicado el martes, 16 de diciembre de 2014 a las 09:10 UTC
Etiquetado como software, calligra

Ya está disponible la primera beta de Calligra 2.9. Concretamente fue anunciado el pasado 14 de diciembre con lo que se mantiene el ritmo de desarrollo previsto de la suite ofimática del futuro de KDE.  Echemos un vistazo a sus novedades.

Disponible la primera beta de Calligra 2.9

Bastantes novedades nos esperan en la siguiente versión de Calligra. Una de ellas ya fue presentada en el blog ya que en el próximo Calligra tendremos las funcionalidades Gemini en todas las aplicaciones . Además, aplicaciones como Kexi o Krita se verán ostensiblemente mejoradas.

A modo de resumen podemos decir que todas las aplicaciones verán resueltos muchos errores y además:

  • Okular, el visor universal de documento de KDE podrá utilizar el motor de Calligra officepara visualizar documentos en formatos abiertos de texto (ODT), MS Word (DOC, DOCX) y WordPerfect (WPD)
  • Nueva funcionalidad Calligra Gemini, que nos permite utilizar de forma más eficiente las aplicaciones Calligra según estemos utilizando ratón o teclado o una pantalla táctil.

  • Kexi:
    • Simplificado y mejorado el sistema de reportes de errores.
    • Paneles laterales más sencillos.

  • Krita: Com siempre, cargado de novedades entre las que destacamos.

¿Qué es Calligra?

Calligra es la apuesta de la Comunidad KDE por crear una suite ofimática moderna y que se adapte a cualquier dispositivo ya sea por el tamaño del mismo (gran o pequeña pantalla) o por el modo de interactuar con él (mediante ratón y teclado o mediante los dedos).

Sus inicios han sido difíciles, pero poco a poco se va viendo como el proyecto se va consolidando y, de hecho, yo ya he elaborado algunos documentos y presentaciones con él, con resultados satisfactorios.

¿Qué aplicaciones ofrece Calligra?

Cómo buena aplicación ofimática, Calligra nos ofrece varias aplicaciones para cubrir todas y cada una de las necesidades que nos generan nuestro que hacer diario.

De esta forma tenemos 10 aplicaciones dentro de la suite Calligra:

  • Words, el procesador de textos.
  • Author, el procesador de textos orientado a los escritores.
  • Sheets, la hoja de cálculo.
  • Stage, la aplicación de las presentaciones.
  • Flow, la aplicación de los diagramas.
  • Kexi, el creador de bases de datos visual
  • Plan, la aplicación de gestión de proyectos
  • Krita, la aplicación para dibujo artístico.
  • Karbon, el programa para dibujos vectoriales.
  • Shapes, el módulo de dibujos para integrarlos en las otras aplicaciones.

 

Prueba y reporta errores

Es el momento de probar la primera beta de Calligra 2.9, para ello basta descargarse el código fuente calligra-2.8.90.tar.xz o el ejecutable para las diversas distribuciones. Y no olvides reportar los errores en KDE Bug Tracking.
