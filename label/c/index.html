Singletons en C++ y alguna nota sobre thread safety (I)

Escrito por Gaspar Fernández en la bitácora Poesía binaria, artículo original
Publicado el lunes, 21 de abril de 2014 a las 08:57 UTC
Etiquetado como singleton, thread, curioso, clase, c, hilos, instancia, seguridad, objeto, c-c, general, cosas-que-damos-por-hechas, single, safety, pthread, patron, objetos, intance, generacion, algoritmos

Antes de nada, comentar que he dividido este post en dos porque vi que se estaba alargando demasiado y se lanzarán uno al día, pondré aquí enlaces a todos los posts.

Muchas veces cuando estamos programando tenemos la necesidad de crear un objeto de una clase determinada, pero éste objeto deberá ser creado una sola vez en nuestra aplicación y debemos evitar a toda costa que pueda ser creado más veces. Podemos pensar en:

  • Una conexión de base de datos para nuestra aplicación. Bueno, a veces necesitamos varias, pero muchas veces sólo necesitamos una, y queremos referirnos a ella en cualquier punto de nuestro programa (y no queremos una variable global ni nada de eso).
  • Un objeto de configuración general del programa. Imaginemos que en el constructor leemos el archivo de configuración y generamos un árbol con las opciones de nuestra aplicación. Dicho árbol no debe ser generado más veces, sólo cuando arrancamos el programa.

BlueFish, un potente editor para desarrollo web.

Publicado el lunes, 10 de marzo de 2014 a las 09:17 UTC
Actualizado el lunes, 10 de marzo de 2014 a las 09:21 UTC

En castellano usamos frecuentemente la frase “Hay muchos peces en el mar”, según la circunstancia. Hoy hablaremos de un jugoso pez azul, que nos puede facilitar mucho las cosas, sobre todo para los desarrolladores web.

En GNU/Linux tenemos un sin fin de editores. Bluefish es un potente editor de texto enfocado al desarrollador.  Cómo características principales tenemos el resaltado de sintaxis, el apoyo a los proyectos, la sangría, la auto-realización, etcétera.  ¿Qué diferencia Bluefish de otros editores con funciones similares?

Partimos de la base que puede editar código para lenguajes como C, C++, Python entre otros. Pero dónde desmarca claramente es para el desarrollo web, ya que tiene soporte para HTML, CSS y PHP, además proporciona una lista todas las etiquetas HTML, y eso facilita mucha las cosas.
