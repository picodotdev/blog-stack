Introduciendo PenTesters Framework (PTF)

jueves, 14 de mayo de 2015, fuente JsiTech, etiquetas seguridad 

Trustedsec ha anunciado el lanzamiento de una herramienta bastante interesante denominada PenTesters Framework (PTF). PTF es una herramienta desarrollada en Python diseñada específicamente para funcionar bajo Distribuciones basadas en Debian o Ubuntu, para convertirlas en distros ideales para las pruebas de penetración. El objetivo de la herramienta es la de instalar diversas herramientas para los Test de Penetracion, y que puedan ser actualizadas de forma sencilla en cualquier distribución. Todo esta organizado de manera que siga los lineamientos del Estándar de ejecución de Test de penetración, PTES ( Penetration Testing Execution Standard), eliminando todo lo que usualmente no se utiliza. ALERTA!! VENOM Vulnerabilidad crítica en QEMU (CVE-2015-3456)

miércoles, 13 de mayo de 2015, fuente JsiTech, etiquetas seguridad qemu 

Ha sido reportado una vulnerabilidad crítica en las plataformas Xen, KVM, y QEMU que le permiten a un atacante romper la seguridad del ambiente de las máquinas virtuales y tomar control del sistema operativo que lo hostea. Se trata de una vulnerabilidad del tipo “Buffer Overflow” que afecta la emulación del controlador del disco Floppy en el componente QEMU en los hipervisores KVM/QEMU y Xen. La vulnerabilidad fue registrada bajo CVE-2015-3456 y ahora es nombrado VENOM (virtualized environment neglected operations manipulation), es considerada de alto impacto. Detalles Un usuario privilegiado puede usar la falla para inhabilitar el Host o ejecutar código arbitrario con el privilegio del proceso QEMU correspondiente a la VM.
