El repo en Git de Exploit Database

martes, 16 de junio de 2015, fuente JsiTech, etiquetas seguridad 

Todos conocemos el proyecto de Exploit Database de Offensive Security, en caso de que no, es un portal donde recopilan exploits públicos o pruebas de concepto de software vulnerables. Estos exploits son desarrollados por Penetration Tester o analistas de seguridad y subidos al portal para uso de toda la comunidad. Todos esos exploits se encuentran en un repositorio en Git, el cual podemos clonar y tener acceso a esos recursos localmente y desde nuestra terminal, además de que es actualizado diariamente. Incluido en el repo esta la utilidad Searchsploit el cual nos permitirá buscar entre los exploits usando uno o varios términos. Para obtener privilegios de administrador existen diversas formas. La más evidente es iniciar sesión en una consola como root, pero eso implica mantenerla abierta hasta que nos aseguremos que no la necesitamos ya más, por lo cual es bastante incómodo como método, además de potencialmente peligroso.
