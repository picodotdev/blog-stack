Ogg, el formato contenedor libre y abierto

lunes, 06 de octubre de 2014, fuente Ocho bits hacen un byte, etiquetas gnu fsf 

Durante muchos años hemos estado acostumbrados a formatos de audio y vídeo restrictivios y privativos. Ogg es un formato radicalmente diferente. Se trata de un formato contenedor libre y abierto, no tiene restricciones de patentes. Con él podremos lograr una calidad de imagen, sonido y audio excelentes. Actualmente Ogg tiene dos variantes, Ogg Vorbis se encarga de codificar audio y Ogg Theora  codifica el vídeo. Ogg se puede reproducir desde muchos programas diferentes, VLC es el recomendado por la Free Software Foundation, como vemos aquí. Existen muchos programos libres aparte de VLC, como Banshee, Amarok y muchos más. No tienes excusas. La FSF y Debian unen fuerzas

martes, 09 de septiembre de 2014, fuente Ocho bits hacen un byte, etiquetas sistemas-operativos debian fsf 

La Free Software Foundation y el proyecto Debian unen fuerzas para ayudar a los usuarios del software libre a encontrar el hardware que necesitan. Ayer, día ocho de septiembre, anunciaron la cooperación para ampliar y mejorar h-node, una base de datos que tiene como fin identificar que dispositivos funcionan con los sistemas operativos completamente libres. Actualmente existen bases de datos con los registros de hardware compatibles con GNU/Linux, en cambio h-node sólo identifica los que son sólo compatibles con software libre, y no necesitan software ni firmware privativo. La base de datos h-node lista tanto los componentes individuales, cómo tarjetas WI-FI y de vídeo, así cómo portátiles o laptops.
