Propiedad padding en CSS

Escrito por Liher en la bitácora El blog de Liher, artículo original
Publicado el martes, 06 de mayo de 2014 a las 11:59 UTC
Actualizado el martes, 06 de mayo de 2014 a las 11:59 UTC
Etiquetado como padding, css, propiedad

En este articulo os voy a hablar de una propiedad CSS llamada “padding“. Es una propiedad bastante sencilla de usar pero suele resultar un poco complicada de asimilar, aunque intentare explicarme lo mejor posible. El concepto en si mismo es el siguiente, con la propiedad “padding” indicamos la distancia que queremos dejar entre en borde de una capa hasta su contenido. Dicho así puede parecer que no es muy útil, pero si que lo es y veréis para que nos va a servir con un ejemplo.

Primero voy a poneros el código de una pagina web muy sencilla en la cual vamos a crear una capa con fondo de color azul, dentro de esa capa azul vamos a poner un texto en color blanco. El código es el siguiente:

<html>

<head>
   <style type="text/css">
      #cuadro
        {
        width: 100px;
        height: 100px;
        background-color: blue;
        color: white;
        }
   </style>
</head>

<body>
   <div id="cuadro">Esto es un ejemplo</div>
</body>

</html>

Posicionamiento de las capas en CSS

Escrito por Liher en la bitácora El blog de Liher, artículo original
Publicado el lunes, 07 de abril de 2014 a las 17:00 UTC
Actualizado el lunes, 07 de abril de 2014 a las 17:03 UTC
Etiquetado como css, posicionamiento, capas

En este articulo os voy a mostrar como podemos posicionar las capas en CSS y lo voy a explicar todo con ejemplos y paso a paso de la mejor forma posible, ya que es un tema importante y que hay que tener muy claro para poder trabajar bien. Antes de empezar con las propiedades propiamente dichas de posicionamiento os voy a mostrar unas etiquetas CSS que nos servirán para ello y que hay que conocer. Os pongo la lista con una pequeña explicación en cada una:

  • top : esta propiedad desplaza una capa tomando como referencia el borde superior de dicha capa. Se usa para desplazar la capa hacia arriba y abajo. Puede tener valores en “px” y en “%” y también pueden ser valores positivos y negativos. Si a una capa le ponemos como propiedad “top: 100px” la bajara 100 pixels respecto al lugar que debería ocupar. Si le ponemos como propiedad “top:-100px” entonces la subirá 100 pixels respecto al sitio que debería ocupar. Esta propiedad solo sirve para los posicionamientos de tipo “relative”, “absolute” o “fixed” que explicare mas abajo.

BlueFish, un potente editor para desarrollo web.

Escrito por ochobitshacenunbyte en la bitácora Ocho bits hacen un byte, artículo original
Publicado el lunes, 10 de marzo de 2014 a las 09:17 UTC
Actualizado el lunes, 10 de marzo de 2014 a las 09:21 UTC
Etiquetado como programacion, web, gnu-linux, css, c, bluefish, desarrollo, php, python, html

En castellano usamos frecuentemente la frase “Hay muchos peces en el mar”, según la circunstancia. Hoy hablaremos de un jugoso pez azul, que nos puede facilitar mucho las cosas, sobre todo para los desarrolladores web.

En GNU/Linux tenemos un sin fin de editores. Bluefish es un potente editor de texto enfocado al desarrollador.  Cómo características principales tenemos el resaltado de sintaxis, el apoyo a los proyectos, la sangría, la auto-realización, etcétera.  ¿Qué diferencia Bluefish de otros editores con funciones similares?

Partimos de la base que puede editar código para lenguajes como C, C++, Python entre otros. Pero dónde desmarca claramente es para el desarrollo web, ya que tiene soporte para HTML, CSS y PHP, además proporciona una lista todas las etiquetas HTML, y eso facilita mucha las cosas.
