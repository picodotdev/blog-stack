Así que vamos a hablar un poco de Docker y en esta serie de entradas (Si. otra serie), veremos de que se trata y aprenderemos como utilizarlo. Que es Docker? Docker es un proyecto Open Source basado en contenedores de Linux, es básicamente un motor de contenedores que usa características del Kernel de Linux como espacios de nombres y controles de grupos para crear contenedores encima del Sistema operativo y automatizar el despliegue de aplicaciones en estos contenedores.
